# Regression Models Dashboard

## Overview
An interactive Streamlit application showcasing 15 different types of regression models with real-time visualizations and comparisons.

## Features
- 15+ regression algorithms
- Interactive visualizations
- Multiple dataset options
- Model comparison tools
- Real-time parameter tuning
- Performance metrics
